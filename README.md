# 🎓 MentorTyme

MentorTyme — це веб-платформа для автоматизації бронювання менторських сесій та керування розкладом між менторами та студентами. Система спрощує процес планування зустрічей завдяки автоматичній генерації доступних слотів та інтеграції з календарями.

---

## ✨ Основні можливості

### 🧠 Розумна система розкладу
Автоматично формує доступні часові слоти на основі розкладу ментора.

### 📅 Інтеграція з Google Calendar
Двостороння синхронізація через OAuth 2.0 — події автоматично створюються та оновлюються в календарях ментора та студента.

### 👥 Рольова система
Окремі сценарії для менторів і студентів із персоналізованими кабінетами.

### 📖 Керування сесіями
Історія бронювань, відстеження сесій та базове управління користувачами.

### 🐳 Контейнеризація (Docker)
Повністю готовий до запуску в будь-якому середовищі через Docker та Docker Compose.

---

## 🛠 Технології

**Backend:** Python, Django, Django ORM  
**Database:** SQLite  
**Frontend:** HTML5, CSS3, Bootstrap  
**Integrations:** Google Calendar API (OAuth 2.0, REST)  
**DevOps:** Docker, Docker Compose  

---

## 🎯 Мета проєкту

Цей проєкт створено для практичного закріплення навичок backend-розробки на Django, роботи з API, автентифікації, моделювання баз даних та реалізації логіки реальних систем планування.
